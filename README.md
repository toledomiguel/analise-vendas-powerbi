# Projeto de BI para Análise de Vendas em E-commerce

Projeto desenvolvido para analisar a performance de vendas, clientes e logística de um e-commerce, utilizando conceitos de Business Intelligence, modelagem de dados e visualização no Power BI.

## Objetivo
Desenvolver um projeto de Business Intelligence para analisar a performance de um e-commerce, com foco em vendas, clientes, produtos e logística.

## Problema de negócio
Este projeto busca responder perguntas importantes para apoiar a tomada de decisão, como:

- Qual é o faturamento total?
- Como as vendas evoluem ao longo do tempo?
- Quais categorias e produtos vendem mais?
- Quais regiões geram mais receita?
- Qual é o ticket médio?
- Há atraso relevante nas entregas?
- O atraso impacta a satisfação do cliente?

## Ferramentas utilizadas
- SQL
- Power BI
- GitHub
- CSV / Excel
- Python (opcional)

## Estrutura do projeto
- `data/raw`: dados brutos
- `data/processed`: dados tratados
- `docs/imagens`: imagens do dashboard e do modelo
- `docs/descricao-negocio`: documentação do projeto
- `sql`: consultas e transformações
- `powerbi`: arquivo do dashboard em Power BI
- `notebooks`: análises exploratórias

## Etapas do projeto
1. Entendimento da base de dados
2. Definição das perguntas de negócio
3. Limpeza e tratamento dos dados
4. Modelagem dos dados
5. Criação de KPIs
6. Desenvolvimento do dashboard
7. Geração de insights
8. Publicação no GitHub
